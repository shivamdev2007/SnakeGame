# 🐍 Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript.

## Features
- Snake movement using arrow keys
- Food generation
- Score tracking
- Wall collision detection
- Restart game functionality

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
